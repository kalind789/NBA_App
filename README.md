I wanted to build an app where people can see what teams that won in the 2025-26 season did well and what they lacked in. Overall I wanted to create a system where I could analyze the reasons that a winning team won vs lost. For example, one game they might have had a really good 3pt% but the next game that same metric might be low. In those outlier games where metrics are lower than the winning average, I wanted to know what other things conpensated for the outlier. This way I was able to get statistics that properly gave insight onto how teams played and what were winning factors.
